# 07 — Full Glove Mount

> *This folder's number is a flexible guide, not a permanent label. Rename or renumber as the real build dictates.*

**Phase:** 7

**Goal:** Mount all five MPU-6050 sensors on the left-hand half-finger glove with proper strain relief. Run the Phase 6 firmware on the real glove.

**"Done" looks like:**
- All five finger sensors readable in a 30-minute wear session with no disconnects.
- Wire runs anchored with Leukoplast tape so no solder pad takes a bend directly.
- Hand-skeleton visualiser still tracks correctly while wearing the glove.
- A photo / short video clip taken as a "Phase 7 complete" artifact.

**Gotcha:** Sensor failures on the glove are almost always wire fatigue at the knuckle — stranded wire and strain-relieved anchoring, not solid-core wire, not tape-then-bend.

**Media discipline:** Capture a "before" photo / serial trace before mounting, and an "after" clip once it works. These are irreplaceable once the layout evolves.
